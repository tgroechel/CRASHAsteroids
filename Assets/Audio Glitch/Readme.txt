How to use.

1. Create a new (or select your) GameObject.
2. Add Component -  Audio > Audio Glitch. 
3. set your "main" clip and "glitch" clip in the inspector.
4. adjust timers.
5. Play.


6. From code call these Functions`

	audio_glitch.Start_Process();
	audio_glitch.Stop_Process();




for more info see the DEMO scene.
